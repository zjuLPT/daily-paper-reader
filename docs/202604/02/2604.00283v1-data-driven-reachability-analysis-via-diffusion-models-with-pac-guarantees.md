## 摘要
我们提出了一种基于数据驱动的非线性动力系统可达性分析框架，该框架无需显式模型。去噪扩散概率模型仅根据轨迹数据学习动力系统的随时间演化的状态分布。预测的可达集合表示为来自重建误差的非一致性评分的一个次级水平集，其阈值通过 Learn Then Test 程序进行校准，以确保排除某个可达状态的概率在高置信度下受到约束。我们在三个非线性系统上进行了实验：受迫Duffing振子、平面四旋翼以及高维反应扩散系统。结果证实经验漏检率始终低于可能近似正确（Probably Approximately Correct，PAC）界限，并且可以扩展到超出经典基于网格和多项式方法所能处理的状态维度。

## Abstract
We present a data-driven framework for reachability analysis of nonlinear dynamical systems that requires no explicit model. A denoising diffusion probabilistic model learns the time-evolving state distribution of a dynamical system from trajectory data alone. The predicted reachable set takes the form of a sublevel set of a nonconformity score derived from the reconstruction error, with the threshold calibrated via the Learn Then Test procedure so that the probability of excluding a reachable state is bounded with high probability. Experiments on three nonlinear systems, a forced Duffing oscillator, a planar quadrotor, and a high-dimensional reaction-diffusion system, confirm that the empirical miss rate remains below the Probably Approximately Correct (PAC) bound while scaling to state dimensions beyond the reach of classical grid-based and polynomial methods.

## 二、方法论概述

### 1. 核心思想

- 利用**去噪扩散概率模型（Denoising Diffusion Probabilistic Model, DDPM）**直接学习动力系统状态的时间分布，不再构建显式动力学模型；
- 以其**重建误差（reconstruction error）**作为“非一致性评分（nonconformity score）”，用于度量样本是否属于可达集合；
- 再通过**Learn-Then-Test (LTT)** 统计校准框架设定阈值，使得被排除的真实可达状态比例在概率意义上受控，从而获得具有**PAC 覆盖保证**的可达集合。

### 2. 方法流程（文字描述）

整体流程如 Fig.1 所示，包括四个阶段：

1. **数据生成**：从未知系统采样 $N$ 条独立轨迹 $\Phi(t; ξ, d)$，得到状态序列；
2. **扩散模型训练**：
   - 构建时间条件扩散模型，定义扩散过程：
     \[
     q(x_\tau | x_0) = \mathcal{N}(\sqrt{\bar{\alpha}_\tau} x_0, (1-\bar{\alpha}_\tau) I)
     \]
   - 训练神经网络 $\epsilon_\theta(x_\tau, \tau, k)$ 去预测噪声；
   - 最小化均方误差 $\mathbb{E}\|\epsilon_\theta - \epsilon\|^2$。
3. **得分计算与校准**：
   - 定义非一致性评分（多噪声步加权误差和）：
     \[
     s(x, k) = \sum_\tau \gamma_\tau \|\epsilon_\theta(x_\tau, \tau, k) - \epsilon_\tau\|^2
     \]
   - 对每一时间步 $k$，使用独立校准集通过 LTT 方法计算阈值 $q_k$，从有限采样中控制漏检率 $\le \alpha$；
4. **可达集合预测**：
   \[
   \hat{R}(t_k) = \{x : s(x, k) \le q_k\}
   \]
   得到具备 PAC 统计保证的预测可达区域。  

### 3. 关键理论结果

- **命题 1（得分与似然的对应关系）**：理论上，理想评分的期望等价于负 ELBO 加上二次项，因此得分次级集近似于数据分布密度的水平集。
- **定理 1（PAC 覆盖定理）**：若校准样本独立同分布，则经 LTT 校准的 $\hat{R}(t_k)$ 同时满足  
  \[
  P(\forall k,\, P_{x\sim μ_k}[x\notin \hat{R}(t_k)] \le \alpha) \ge 1-\delta
  \]
  即漏检概率的 PAC 上界。
- **推论（几何覆盖）**：给定正初始密度及可微变换假设，可将分布覆盖提升为可达集的**体积（几何）上限**，并在耗散系统中随时间收紧。

### 2. 数据与校准设置

- 阈值网格划分为 2000 个候选 λ；
- 使用 LTT 框架基于 Hoeffding–Bentkus p-value 选定最小满足置信度的阈值。

### 3. 对比基线方法

- **Normalizing Flow (NF)**：用负对数似然作评分；
- **Christoffel Function 方法**：[Devonport et al., CDC 2021] 多项式子层表示；
- **Variational Autoencoder (VAE)**：基于重构 ELBO 的异常分数；
- 所有基线除 Christoffel 外均纳入 LTT 校准。

### 4. 评估指标

- **IoU（交并比）**与**Precision（精确度）**用于低维；
- **误检率（FNR）**检验 PAC 覆盖；
- 在高维场景下使用**接收率–扰动曲线**与**得分分布分离度**作为替代衡量。
