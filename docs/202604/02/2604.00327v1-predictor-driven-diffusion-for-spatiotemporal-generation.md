## 摘要
多尺度空间结构使时间预测变得复杂，因为小尺度空间波动会影响大尺度演化，而解析所有尺度往往难以实现。标准扩散模型无法有效解决这一问题，因为它们在所有傅里叶模式上施加统一的衰减。我们提出了预测器驱动扩散（Predictor-Driven Diffusion）框架，将基于重整化群的空间粗粒化与时间动力学的路径积分形式相结合。前向过程通过尺度依赖的拉普拉斯阻尼配合加性噪声，生成由扩散尺度 λ 索引的粗粒场层次结构。训练过程最小化由数据和预测器诱导的路径密度之间的 Kullback–Leibler 散度，从而得到关于时间导数的简单回归损失。所得预测器能够捕捉被消除的小尺度分量对大尺度演化的统计性影响。一个关键见解是，相同的预测器为逆向 λ 采样提供路径评分，从而将仿真、无条件生成和超分辨率统一为单一框架。我们的统一方法通过两个多尺度湍流系统的实验得到了验证。

## Abstract
Multiscale spatial structure complicates temporal prediction because small-scale spatial fluctuations influence large-scale evolution, yet resolving all scales is often intractable. Standard diffusion models do not address this problem effectively since they apply uniform decay across all Fourier modes. We propose Predictor-Driven Diffusion, a framework that combines renormalization-group-based spatial coarse-graining with a path-integral formulation of temporal dynamics. The forward process applies scale-dependent Laplacian damping together with additive noise, producing a hierarchy of coarse-grained fields indexed by diffusion scale $λ$. Training minimizes the Kullback-Leibler divergence between data-induced and predictor-induced path densities, leading to a simple regression loss on temporal derivatives. The resulting predictor captures how eliminated small-scale components statistically influence large-scale evolution. A key insight is that the same predictor provides a path score for reverse-$λ$ sampling, unifying simulation, unconditional generation, and super-resolution in a single framework. Our unified approach is validated through experiments on two multiscale turbulent systems.

## 1. 研究核心问题与背景

- **问题定义**：  
  许多自然与工程系统具有显著的多尺度结构——小尺度扰动会影响大尺度动力演化。例如，气象过程中的云形成影响天气模式。  
  这类系统的预测困难在于：同时解析所有空间尺度的计算量极高，而忽略小尺度又会丢失其对系统演化的统计影响。

- **现有方法局限**：  
  标准扩散模型（Diffusion Models）在所有傅里叶模式上施加统一衰减，因此未能显式地区分并建模不同尺度结构。它们的“多尺度性”只是噪声层级上的隐含特性，缺乏物理解释。

- **研究动机**：  
  希望借助物理学中的**重整化群（Renormalization Group, RG）**思想，把空间尺度结构显式地引入扩散模型，让它既能保持因果性（时间方向不可反向），又能充分利用多尺度信息。

## 2. 方法论：预测器驱动扩散框架

### 核心思想

- 将**空间尺度**与**物理时间**作为两个独立的轴：  
  - 扩散尺度（λ）控制空间粗粒化层次。  
  - 物理时间（t）保持因果演化。
- 在每个空间尺度 λ 下，定义一个由神经网络拟合的**预测器（drift function）**，以产生时序动力。
- **KL 散度最小化**训练：学习一个预测器，使得预测器诱导的路径概率分布与真实数据路径分布尽可能一致。

### 技术构成

- **空间粗化（Forward RG Diffusion）**：  
  \[
  ∂_λ u_λ = α∇_x^2 u_λ + βη_λ
  \]
  - 基于拉普拉斯算子 \(∇^2_x\)，进行尺度依赖的阻尼。  
  - βηλ 是白噪声项，用于保持统计一致性。  
  - 得到空间上逐步粗化的层级 \(u_λ\)。

- **时间动力建模（Predictor Governing Equation）**：  
  \[
  ∂_t u_λ = f_{θ,λ}(u_λ) + σ_λξ
  \]
  - 其中 \(f_{θ,λ}\) 为可学习的神经网络预测器。  
  - σλξ 表示由于粗化消除的小尺度成分带来的扰动。

- **训练目标（KL 散度 → 回归损失）**：  
  最小化数据路径分布 qλ 与预测器路径分布 pλ 的 KL 散度：
  \[
  D_{KL}(q_λ‖p_λ) = \frac{1}{2σ_λ^2} E_{q_λ}[(∂_t u_λ - f_{θ,λ}(u_λ))^2] + const
  \]
  → 等价于简单的时间导数回归损失。

- **统一推理流程**：
  - **仿真（Simulation）**：固定 λ，按照时间方程前向积分。  
  - **生成（Generation）**：反向积分 λ（从大尺度到小尺度），使用路径积分求得 score。  
  - **超分辨（Super-Resolution）**：将粗粒结果逆向 λ 采样回细尺度。

### 算法结构（文字版）

1. 使用拉普拉斯算子计算空间粗化的算子 \(C_λ\) 与协方差矩阵 \(Σ_λ\)。  
2. 对多个 λ 值抽样训练，最小化时间导数回归损失。  
3. 经训练后，同一个预测器网络既可以模拟时间演化，又能实现生成与超分辨。  
4. 逆 λ 采样采用路径积分的数值积分（带 Λ 方向的 SDE ），并在 Fourier 空间中进行指数时间差分更新。
